API documentation

Here's a list of the methods used.

Books:

Users:

GET /books/list

Returns a list of books added by the user logged in.

Endpoint URL

http://127.0.0.1:8090/books/list

Example request:

curl -X GET http://127.0.0.1:8090/books/list

Example response:

{
"books":[
    {
    "title": "The Great Gatsby",
    "author": "F. Fitzgerald",
    "review": "Nice",
    "userfname": "Adriana",
    "userlname": "Tihan"
    }
],
"users" : [
    {
    "fname" : "Name",
    "lname" : "Name"
    }
]
}

Response fields:

Name Type Description
books.title string The title of the book, as entered by user.
books.author string The author of the book, as entered by user.
books.review string The review the user gave of the book.
books.userfname string First name of the user who added the book.
books.userlname string Last name of the user who added the book.
users.fname string First name of current user.
users.lname string Last name of current user.
